🏥 Green City Hospital & Diagnostic Center Website

A modern, responsive, and visually appealing hospital website built using HTML5 and CSS3. This project is designed to provide a professional online presence for healthcare institutions, featuring service highlights, department information, appointment booking, and patient-focused design.
